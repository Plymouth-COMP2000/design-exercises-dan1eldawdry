# Restaurant Android Application

This project is an Android app developed using Android Studio.  
It supports guest and staff users with different functionality, including menu viewing, reservation management and notifications.

## Third Party Resources Used

- Volley – HTTP networking for REST API communication  
  https://developer.android.com/training/volley

- Gson – JSON parsing for API responses  
  https://github.com/google/gson

- Material Components for Android – UI components and input layouts  
  https://material.io/develop/android

- AndroidX Libraries – Modern Android support libraries  
  https://developer.android.com/jetpack/androidx

## How to Run

Open the project in Android Studio, sync Gradle and run on an emulator or physical device.
